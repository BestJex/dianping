# dianping
介绍： 基于elasticSearch + Spark的智能搜索和推荐系统

## 基础服务搭建
### 品类管理服务
- 新增品类
- 查询品类

### 门店服务
- 门店创建
- 门店地理位置（基于lbs 地理位置接入）
- 门店查询
