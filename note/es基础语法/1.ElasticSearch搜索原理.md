# ES是什么？

- 独立的网络上的一个或一组进程节点或者说一个应用或中间件
- 它对外提供搜索服务(http 或 tarnsport协议)
- 对内就是一个**搜索数据库**

## 名称定义

- 索引 = 数据库
- 类型 = 表(最新的ElasticSearch7废弃了类型的概念)
- 文档 = 行数据

## 关系型数据库对应于ElasticSearch

|  关系型数据库   | ElasticSearch  |
|  ----  | ----  |
| Database(数据库)  | Index(索引) |
| Table(表)  | Type(类型) |
| Row(行) | Document(文档) |
| Column(列) | Field |
| Schema(表结构) | Mapping |
| Index(索引) | 所有的字段都可以做索引 |
| SQL | Query DSL |

### 索引

- 搜索中的数据库或表定义
- 构建文档时候的索引创建
  
## 分词

- 搜索是以词为单位做的最基本单搜索单元
- 依靠**分词器**构建分词
- 用分词器构建**倒排索引**
